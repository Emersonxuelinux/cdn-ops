# cdn_ops
    cdn运维管理系统，本系统主要依赖于django 框架，同时边缘服务器初始化的时候还使用django-rq,异步任务队列，使用salt-ssh 调用salt 初始化model，初始化边缘设备，然后
    成功后, 会回调web系统；域名下发生效以及刷新预加载等任务都差不多原理;
